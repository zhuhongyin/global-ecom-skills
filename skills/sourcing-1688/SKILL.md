# 1688 Sourcing

## 概述

此Skill用于在1688平台查找产品货源，帮助找到优质工厂供应商，获取有竞争力的批发价格。

## 使用场景

当用户需要：
- 查找产品的1688货源
- 获取工厂批发价格
- 评估供应商资质
- 计算采购成本

## 执行步骤

### 步骤1：关键词搜索

使用产品关键词在1688搜索：

- 搜索URL：https://s.1688.com/selloffer/offer_search.htm?keywords=[关键词]
- 使用中文关键词搜索效果更好
- 可添加"工厂"、"源头"等词筛选

### 步骤2：供应商筛选

筛选优质供应商：

| 筛选条件 | 说明 |
|---------|------|
| 超级工厂 | 优先选择 |
| 实力商家 | 次优先 |
| 诚信通年限 | ≥3年为佳 |
| 响应速度 | 快速响应 |
| 复购率 | 高复购率 |

### 步骤3：产品信息提取

对每个产品提取：

| 字段 | 说明 |
|------|------|
| 产品名称 | 产品标题 |
| 起批价 | 最低批发价 |
| 起订量 | MOQ |
| 规格 | 尺寸、材质等 |
| 发货地 | 产地 |
| 供应商 | 工厂名称 |
| 诚信通年限 | 经营年限 |
| 主营类目 | 工厂专长 |

### 步骤4：价格分析

分析价格区间：

```
价格分析：
- 最低价: ¥[价格]
- 平均价: ¥[价格]
- 最高价: ¥[价格]
- 建议采购价: ¥[价格]
```

### 步骤5：供应商评估

评估供应商质量：

| 评估维度 | 权重 | 评分标准 |
|---------|------|---------|
| 价格竞争力 | 30% | 是否低于均价 |
| 工厂资质 | 25% | 超级工厂/实力商家 |
| 诚信通年限 | 15% | 年限越长越好 |
| 响应速度 | 15% | 24小时内响应 |
| 复购率 | 15% | 高复购率为佳 |

### 步骤6：输出分析报告

```

- 搜索关键词: [关键词]

- 起批价: ¥[价格]

[基于分析的采购建议]

### 找货话术
给采购员的精准话术：
"您好，我需要采购[产品名称]，要求[材质/规格]，预计首批[数量]件，请报最优价格和交期。"
```

## 成本计算

### 完整成本构成

```
产品成本 = 1688起批价
国内履约费 = ¥3.50（固定）
总成本 = 产品成本 + 国内履约费

收入计算：
Temu售价 = $[价格]
回款 = Temu售价 × 45%
收入(人民币) = 回款 × 7.2

净利润 = 收入(人民币) - 总成本
```

### 利润红线

**净利润必须 > ¥5.00**

## 注意事项

1. **避坑指南**
   - 不要只看价格最低
   - 注意产品质量
   - 确认是否含税
   - 了解退换货政策

2. **优质工厂特征**
   - 有自有生产线
   - 支持定制
   - 有出口资质
   - 响应及时

3. **谈判技巧**
   - 量大可议价
   - 长期合作有优惠
   - 可要求样品确认质量
