# Temu Competitor Analyzer

## 概述

此Skill用于分析Temu平台的竞品情况，评估市场竞争程度，确定定价策略，判断产品是否有上架机会。

## 使用场景

当用户需要：
- 分析Temu上某类产品的竞争情况
- 了解竞品定价策略
- 评估市场饱和度
- 确定产品是否有上架机会

## 执行步骤

### 步骤1：搜索Temu竞品

使用Web搜索工具，在Temu平台搜索目标产品：

- 搜索URL：https://www.temu.com/search?q=[产品关键词]
- 记录搜索结果数量
- 分析前20个竞品

### 步骤2：竞品信息提取

对每个竞品提取以下信息：

| 字段 | 说明 |
|------|------|
| 产品名称 | 竞品标题 |
| 价格 | 当前售价 |
| 原价 | 划线价（如有） |
| 销量 | 已售数量 |
| 评分 | 用户评分 |
| 评论数 | 评论数量 |
| 店铺 | 卖家店铺名 |
| 发货地 | 发货地点 |

### 步骤3：竞争程度评估

计算以下指标：

1. **市场饱和度**
   - 搜索结果数量
   - 同质化程度（相似产品占比）

2. **价格分析**
   - 最低价（卷王价）
   - 最高价
   - 平均价
   - 价格分布

3. **销量分析**
   - 头部卖家销量
   - 中腰部销量
   - 新品销量

### 步骤4：机会评估

根据以下标准评估上架机会：

| 评估维度 | 有机会 | 需谨慎 | 不建议 |
|---------|--------|--------|--------|
| 搜索结果 | <1000 | 1000-5000 | >5000 |
| 同质化 | 低 | 中 | 高 |
| 价格空间 | 有利润 | 刚好盈亏 | 亏损 |
| 头部销量 | <1000 | 1000-5000 | >5000 |

### 步骤5：输出分析报告

```
## 📊 Temu竞品分析报告

### 搜索概况
- 搜索关键词: [关键词]
- 搜索结果数: [数量]

### 价格分析
- 卷王价（最低价）: $[价格]
- 平均价: $[价格]

### 竞争程度
- 市场饱和度: [低/中/高]
- 同质化程度: [低/中/高]

### 上架机会评估
- 综合评分: [1-10分]

```

## 定价策略

### V4.1核价模型

```
Temu卷王价 = 竞品最低价
你的回款预估 = 卷王价 × 45%
你的收入(人民币) = 回款预估 × 7.2
```

### 利润计算

```
总成本 = 1688拿货价 + ¥3.50(国内履约费)
净利润 = 收入(人民币) - 总成本
```

**铁律：净利润必须 > ¥5.00**

## 注意事项

1. 关注Temu全托管模式要求
2. 注意JIT模式的库存要求
3. 避免与头部卖家正面竞争
4. 寻找差异化机会
