# Amazon Trend Analyzer

## 概述

此Skill用于分析亚马逊Movers & Shakers（飙升榜）数据，帮助识别近期热销和有潜力的产品趋势。

## 使用场景

当用户需要：
- 了解亚马逊当前热销产品趋势
- 获取飙升榜产品信息
- 分析产品市场潜力
- 为跨境电商选品提供数据依据

## 执行步骤

### 步骤1：访问亚马逊飙升榜

使用Web搜索和抓取工具，访问以下亚马逊飙升榜页面：

- 美国站：https://www.amazon.com/Best-Sellers/zgbs
- 飙升榜：https://www.amazon.com/movers-and-shakers

### 步骤2：提取产品信息

对每个产品提取以下信息：

| 字段 | 说明 |
|------|------|
| 产品名称 | 完整的产品标题 |
| ASIN | 亚马逊产品标识码 |
| 当前价格 | 美元价格 |
| 原价 | 划线价（如有） |
| 飙升排名 | Movers & Shakers排名 |
| 类目 | 产品所属类目 |
| 评分 | 星级评分 |
| 评论数 | 评论数量 |
| 飙升幅度 | 排名上升百分比 |

### 步骤3：分析产品趋势

分析以下维度：

1. **热度指标**
   - 飙升幅度是否超过100%
   - 是否为季节性产品
   - 是否为新品

2. **竞争分析**
   - 评论数量（<100为蓝海机会）
   - 评分情况（4.0以上为佳）

3. **价格区间**
   - 适合Temu定价的价格区间（$10-$50最佳）

### 步骤4：输出分析报告

按以下格式输出：

```
## 🔥 亚马逊飙升榜分析报告

### 热门品类趋势
[分析当前热门品类]

### 推荐关注产品（Top 5）

#### 1. [产品名称]
- ASIN: [ASIN]
- 价格: $[价格]
- 飙升幅度: [百分比]
- 类目: [类目]
- 评分: [星级] ([评论数]条评论)
- 推荐理由: [简要说明]

[重复2-5]

### 选品建议
[基于数据的选品建议]
```

## 注意事项

1. 避免选择以下类型产品：
   - 需要特殊认证的产品（医疗器械、婴儿用品等）
   - 品牌侵权风险高的产品
   - 体积过大或重量过重的产品

2. 优先选择：
   - 轻小件产品（运费低）
   - 组合套装机会
   - 差异化设计空间大的产品

按照上述步骤执行分析，输出完整的飙升榜分析报告。
