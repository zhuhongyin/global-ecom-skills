# 4Supply Platform Analyzer

## 概述

此Skill用于分析乐歌公司4Supply B2B平台的产品情况，帮助了解该平台的选品机会和市场定位。

## 平台简介

4Supply是乐歌(Loctek)公司的B2B采购平台，主要经营：
- 办公家具
- 站立式办公桌
- 显示器支架
- 人体工学产品
- 智能家居产品

## 使用场景

当用户需要：
- 了解4Supply平台热销产品
- 分析平台价格水平
- 评估产品上架机会
- 对比其他平台定价

## 执行步骤

### 步骤1：访问平台

访问4Supply平台：https://www.4supply.com/home

### 步骤2：产品信息收集

收集以下信息：

| 字段 | 说明 |
|------|------|
| 产品名称 | 产品标题 |
| 产品类目 | 所属分类 |
| 批发价格 | B2B价格 |
| MOQ | 最小起订量 |
| 产品规格 | 尺寸、材质等 |
| 库存状态 | 是否有货 |

### 步骤3：热销品类分析

分析平台热销品类：

1. 站立式办公桌
2. 显示器支架
3. 办公桌配件
4. 人体工学椅
5. 智能升降系统

### 步骤4：价格对比

与亚马逊、Temu价格对比：

```
价格对比表：
| 产品 | 4Supply价格 | 亚马逊价格 | Temu价格 | 利润空间 |
|------|------------|-----------|----------|---------|
| ... | ... | ... | ... | ... |
```

### 步骤5：输出分析报告

```
## 🏢 4Supply平台分析报告

### 平台概况
- 平台定位: B2B批发
- 主营类目: [类目列表]
- 热销品类: [Top 5]

### 产品分析

#### 热销产品Top 5
1. [产品名称]
   - 批发价: $[价格]
   - MOQ: [数量]
   - 亚马逊对标价: $[价格]
   - 利润空间: [百分比]

[重复2-5]

### 选品建议
[基于分析的建议]
```

## 注意事项

1. 4Supply是B2B平台，价格通常低于零售
2. 注意MOQ要求
3. 考虑物流成本
4. 关注产品认证要求

## 与其他平台的协同

### 选品流程

```
亚马逊飙升榜 → Temu竞品分析 → 4Supply价格对比 → 选品决策
```

### 定价策略

在4Supply上架时：
- 参考亚马逊零售价的50%-70%
- 确保B2B利润空间
- 考虑批量折扣

**用户输入：**
> 分析4Supply平台上站立式办公桌的产品情况

**输出：**
按照上述步骤执行分析，输出完整的平台分析报告。
