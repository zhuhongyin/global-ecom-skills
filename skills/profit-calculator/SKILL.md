# Profit Calculator

## 概述

此Skill用于计算跨境电商产品的利润，严格按照V4.1核价模型进行核算，确保选品决策有数据支撑。

## V4.1核价模型

### 核心公式

```
┌─────────────────────────────────────────────────────────┐
│                    V4.1 利润计算模型                      │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  Temu卷王价 = 竞品最低价 ($)                             │
│                                                         │
│  回款预估 = 卷王价 × 45% ($)                             │
│                                                         │
│  收入(人民币) = 回款预估 × 7.2 (¥)                       │
│                                                         │
│  总成本 = 1688拿货价 + ¥3.50 (¥)                         │
│                                                         │
│  净利润 = 收入(人民币) - 总成本 (¥)                       │
│                                                         │
│  【铁律】净利润必须 > ¥5.00                              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 参数说明

| 参数 | 说明 | 来源 |
|------|------|------|
| Temu卷王价 | Temu同类竞品最低价 | temu-competitor-analyzer |
| 45% | 平台扣除比例 | 平台规则 |
| 7.2 | 美元兑人民币汇率 | 实时汇率 |
| 1688拿货价 | 工厂批发价 | sourcing-1688 |
| ¥3.50 | 国内履约费 | 固定成本 |

## 使用场景

当用户需要：
- 计算产品利润
- 判断产品是否值得上架
- 比较不同产品的盈利能力
- 制定定价策略

## 执行步骤

### 步骤1：收集数据

收集以下必要数据：

| 数据项 | 来源 | 示例 |
|--------|------|------|
| 产品名称 | 用户输入 | 桌垫 |
| Temu卷王价 | Temu分析 | $12.99 |
| 1688拿货价 | 1688分析 | ¥25.00 |

### 步骤2：计算收入

```python
temu_price = float(input("Temu卷王价($): "))
payment_rate = 0.45
exchange_rate = 7.2

payment_usd = temu_price * payment_rate
income_cny = payment_usd * exchange_rate

print(f"回款预估: ${payment_usd:.2f}")
print(f"收入(人民币): ¥{income_cny:.2f}")
```

### 步骤3：计算成本

```python
price_1688 = float(input("1688拿货价(¥): "))
fulfillment_fee = 3.50

total_cost = price_1688 + fulfillment_fee

print(f"总成本: ¥{total_cost:.2f}")
```

### 步骤4：计算利润

```python
net_profit = income_cny - total_cost

print(f"净利润: ¥{net_profit:.2f}")

if net_profit > 5.00:
    print("✅ GO - 利润达标，建议上架")
else:
    print("❌ PASS - 利润不足，不建议上架")
```

### 步骤5：输出核价报告

```
## 💰 利润核价报告

- 产品名称: [名称]
- 计算时间: [时间]

### 成本计算

### 最终判定
[✅ GO / ❌ PASS]

### 建议
[基于计算的建议]
```

## 利润率分析

### 利润率等级

| 净利润 | 利润率 | 等级 | 建议 |
|--------|--------|------|------|
| >¥20 | >30% | 优秀 | 强烈推荐 |
| ¥10-20 | 20-30% | 良好 | 推荐上架 |
| ¥5-10 | 10-20% | 及格 | 可以考虑 |
| <¥5 | <10% | 不达标 | 不建议 |

### 敏感性分析

当利润不达标时，分析可调整空间：

1. **降低1688采购价**
   - 寻找更低价供应商
   - 增加采购量议价

2. **提高Temu售价**
   - 差异化定价
   - 组合套装

3. **优化成本**
   - 减少包装成本
   - 优化物流

## 批量计算

支持批量计算多个产品：

```
| 产品 | Temu价 | 1688价 | 收入 | 成本 | 净利润 | 判定 |
|------|--------|--------|------|------|--------|------|
| A | $15 | ¥30 | ¥48.6 | ¥33.5 | ¥15.1 | ✅ GO |
| B | $10 | ¥40 | ¥32.4 | ¥43.5 | -¥11.1 | ❌ PASS |
| C | $20 | ¥50 | ¥64.8 | ¥53.5 | ¥11.3 | ✅ GO |
```

## 注意事项

1. **汇率波动**
   - 使用实时汇率
   - 预留汇率波动空间

2. **隐藏成本**
   - 退换货成本
   - 质量问题损耗
   - 平台促销费用

3. **保守估算**
   - 使用卷王价而非理想价格
   - 预留利润空间
